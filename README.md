## Errata

###Pagina 16:
Dentro do programa da página 15 que continua nesta página. Trocar o **-** pelo **=**. 

####Errado:
	c - getchar();

####Certo:
	c = getchar();

###Pagina 28:
Dentro do programa, tem uma funçao **copia** que possui um **while**, onde está faltando um **)**.

####Errado:
	while ((para [i] = de [i] != '\0')

####Certo:
	while ((para [i] = de [i] ) != '\0')
